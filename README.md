# HRM Management Software

This is a custom-built HRM (Human Resource Management) system designed to manage employees, attendance, roles, leaves, and payroll operations.

The system includes an admin dashboard with role-based access, allowing organizations to efficiently manage internal workflows and employee records.

This project demonstrates real-world business process automation and backend system development.
